# 🍕 My Pizza App  

A modern, responsive web application for browsing and ordering pizza online. Built with **Next.js**, **React**, and **Tailwind CSS**, it delivers a smooth and interactive shopping experience.  

---

## ✨ Features  
- 🛒 **Add to Cart** – Select pizzas and instantly add them to your cart.  
- ❌ **Remove Items** – Easily delete products from the cart.  
- 🔍 **Search** – Quickly find your favorite pizza by name.  
- 📂 **Category Filtering** – Switch between Pizza, Burger, Fries, and more.  
- 💰 **Order Summary** – Live-updating checkout panel with total price.  
- 📱 **Responsive Design** – Optimized for desktop and mobile.  
- ⚡ **Fast UI** – Powered by dynamic rendering and client-side state management.  

---

## 🛠️ Tech Stack  
- **Next.js (App Router)** – SSR & routing  
- **React** – UI components  
- **Tailwind CSS** – Styling  
- **Context API** – Cart state management  
- **MUI Icons** – Modern UI elements

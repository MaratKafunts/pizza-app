🍕 My Pizza App
A modern, responsive web application for browsing and ordering pizza online. Built with Next.js, React, and Tailwind CSS, it delivers a smooth and interactive shopping experience.
✨ Features

🛒 Add to Cart – Select pizzas and instantly add them to your cart.
❌ Remove Items – Easily delete products from the cart.
🔍 Search Functionality – Quickly find your favorite pizza by name.
📂 Category Filtering – Switch between categories (e.g. Classic, Vegetarian, Specials) for easier browsing.
💰 Order Summary – View a live-updating checkout panel with total price.
📱 Responsive Design – Optimized for desktop and mobile screens.
⚡ Fast UI – Powered by dynamic rendering, client-side state management, and optimized assets.

🛠️ Tech Stack
Next.js (App Router) for server-side rendering and routing.
React for building UI components.
Tailwind CSS for styling.
Context API for cart state management.
MUI Icons for modern UI elements.
